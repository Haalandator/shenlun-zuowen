# shenlun-zuowen · 申论大作文多角色写作流水线

一个 Claude 用户技能（Skill）：以"审材立意 → 提纲 → 起草 → 质检 →
阅卷 → 定稿"的多角色流水线写申论议论文，用户作为第一作者，全程人工把关。当前仅为测试版本，测试进度（7/70）。

## 三种入口
1. 全流程：贴出题干＋给定资料，说"申论作文"；
2. 单角色点名：如"只出立意请示单""只审这份提纲"；
3. 批改模式：提供成稿，直接模拟阅卷评档。

## 目录结构
- SKILL.md　　　　主规范（协作宪章／共同规范／闸口流程）
- references/　　 六个角色卡＋架构师＋成稿检验单＋事故档案
- scripts/　　　　zishu.py 字数核验｜cipin.py 词频｜shuangyao.py 双钥枚举

## 安装
在 Claude 中作为用户技能上传本文件夹（保持目录结构），
